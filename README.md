
# 🗳️ Decoding the 2026 Tamil Nadu Assembly Election

## 📌 Headline Finding

Tamil Nadu’s 2026 election reflects a **statewide voter shift—driven by constituency flips, regional swings, and growing vote share momentum.**

---

## 📖 Project Overview

This project analyzes Tamil Nadu Assembly Election results (2021 vs 2026) using constituency-level data to uncover how voter behavior, party performance, and regional dynamics have evolved.

The analysis focuses on identifying **patterns**, not causes, using data-driven insights.

---

## 🎯 Objectives

* Identify constituencies where winners changed (Flip Analysis)
* Analyze regional voting patterns
* Measure vote share shifts across major parties
* Understand emerging political momentum

---

## 📊 Key Insights

🔄 **Flip Story**

* A significant number of constituencies changed winners
* Indicates active voter switching and reduced loyalty

🗺️ **Geographic Shift**

* Regional variations observed in voting patterns
* Metro regions showed stronger swings

📊 **Vote Share Movement**

* Emerging parties gained vote share beyond seats won
* Established parties show mixed performance trends

📈 **Momentum Insight**

* Vote share growth indicates future electoral potential
* Several constituencies show rising support patterns

## 🔁 How to Reproduce

1. Load datasets into Power BI
2. Create relationships using constituency ID
3. Build calculated measures:

   * Vote Share %
   * Vote Share Change
   * Flipped Constituencies
4. Develop visuals:

   * Sankey Diagram (Party Shift)
   * Map (Geographic Analysis)
   * Bar Charts (Regional Comparison)
   * KPI Cards (Swing, Growth)
